#  String Manipulation Tool (Python)

A simple Python command-line application that allows users to perform basic string operations.

##  Features

- Convert text to lowercase
- Convert text to uppercase
- Slice a string
- Count string length
- Loop through characters
- Search for a word or letter
- Reverse a string

## 🛠️ Technologies

- Python 3

##  How to Run

1. Clone the repository
2. Open the file in your terminal
3. Run:

```bash
